# Local Development

### Requirement
- `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Set enviroment

Copy .env.example to .env (do not rename) and fill the fields

## Run Tests

`npx hardhat run --network ccnbeta ./test/main.js`
